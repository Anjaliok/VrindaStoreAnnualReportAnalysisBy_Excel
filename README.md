# 📊 Vrinda Store Sales Analysis – Excel Data Analytics Project

📌 Project Overview

This project focuses on analyzing **Vrinda Store’s sales data** using **Microsoft Excel** to generate meaningful business insights through data cleaning, processing, and interactive dashboard creation.
The goal of this project is to transform raw sales data into an **interactive analytical dashboard** that helps businesses understand customer behavior, sales performance, and order trends.

🎯 Project Objectives

* Analyze overall sales performance
* Identify top-performing states and customers
* Understand sales trends across months
* Analyze gender-wise and age-group purchases
* Track order status and category performance
* Build an interactive business dashboard

🛠️ Tools & Technologies Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers & Filters
* Data Cleaning Techniques
* Excel Formulas & Functions
* Data Visualization

📂 Dataset Description

The dataset contains e-commerce sales information such as:

* Order ID
* Customer Gender & Age Group
* State & Region
* Product Category
* Sales Amount
* Order Status
* Channel (Amazon, Flipkart, Myntra, etc.)
* Order Date

This data helps analyze customer purchasing patterns and sales distribution.

1. Data Collection

 📊 Dashboard Features

✅ Monthly Sales Trend Analysis
✅ Sales by State
✅ Gender-wise Sales Distribution
✅ Age Group Analysis
✅ Order Channel Performance

✅ Interactive Filters & Slicers

The dashboard allows users to dynamically explore sales insights.

 📈 Key Insights

* Majority of purchases were made by **female customers**
* Certain states contributed significantly to total revenue
* Adult age group showed highest buying behavior
* Online marketplaces played a major role in sales generation
* Seasonal trends impacted monthly sales performance
